# Unraid Dashboard Tile Stub

This is a minimal Unraid 7.2 plugin that adds a custom tile to the dashboard.

## Features
- Adds a tile with a title, icon, and placeholder content
- Easy to extend with system stats or custom HTML

## Installation
1. Copy `DashboardTileStub.plg` to your Unraid server.
2. Install via the Unraid plugin manager.
3. The tile will appear on your dashboard.
